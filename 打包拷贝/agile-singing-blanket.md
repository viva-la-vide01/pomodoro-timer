# 桌面番茄钟软件 - 实现计划

## Context
用户需要一个桌面番茄钟（Pomodoro Timer）软件。项目目录为空，从零开始构建。选择 Electron 框架，因为它能构建跨平台桌面应用，配合 HTML/CSS/JS 可以做出精美的 UI。

## 技术选型
- **Electron** - 桌面应用框架，跨平台（Windows/macOS/Linux）
- **HTML/CSS/JS** - 前端界面，无需额外框架，保持轻量
- **Node.js** - 后端逻辑（通知、系统托盘、设置持久化）

## 功能设计

### 核心功能
1. **倒计时器** - 清晰的圆形进度条 + 数字显示
2. **三种模式** - 专注（25分钟） / 短休息（5分钟） / 长休息（15分钟）
3. **开始/暂停/重置** - 标准控制按钮
4. **自动切换** - 专注结束 → 短休息 → 专注 → ...，每4个专注后进入长休息
5. **桌面通知** - 计时结束时弹出系统通知
6. **音效提示** - 计时结束播放提示音
7. **会话计数** - 显示今日完成番茄数
8. **任务列表** - 简单的待办事项，关联当前专注任务

### 进阶功能
9. **系统托盘** - 最小化到托盘，托盘显示剩余时间
10. **窗口置顶** - 可切换置顶模式
11. **自定义时长** - 设置页面修改各模式时长
12. **历史记录** - 记录每日完成数量

## 文件结构

```
pomodoro-timer/
├── package.json
├── main.js              # Electron 主进程（窗口、托盘、通知）
├── preload.js           # 预加载脚本（IPC 桥接）
├── renderer/
│   ├── index.html       # 主界面
│   ├── style.css        # 样式
│   └── renderer.js      # 渲染进程逻辑（计时器、UI 更新）
└── assets/
    └── bell.mp3         # 提示音（用代码生成，无需外部文件）
```

## 实现细节

### 主进程 (main.js)
- 创建无边框或简约窗口（320×480，可调整大小）
- 系统托盘图标 + 右键菜单（显示/退出）
- 窗口置顶切换
- 使用 `Notification` API 发送桌面通知
- 通过 IPC 与渲染进程通信（控制计时器状态）
- 使用 `electron-store` 或手动 JSON 文件保存设置

### 渲染进程 (renderer.js)
- 使用 `setInterval` 实现倒计时（精度1秒）
- Canvas 绘制圆形进度条
- 状态机管理：IDLE → RUNNING → PAUSED → BREAK → IDLE
- 自动模式切换逻辑
- 任务列表 CRUD

### UI 设计 (style.css)
- 暗色主题背景，清新简洁
- 大号倒计时数字居中显示
- SVG 圆形进度环
- 底部控制按钮（开始/暂停、跳过、重置）
- 顶部模式标签切换
- 使用 CSS 变量统一配色

## 实施步骤

1. **初始化项目** - 创建 package.json，安装 Electron
2. **主进程** - 编写 main.js（窗口创建、托盘、IPC）
3. **预加载** - 编写 preload.js（安全暴露 API）
4. **界面** - 编写 HTML + CSS（番茄钟 UI）
5. **逻辑** - 编写 renderer.js（计时、状态管理、任务列表）
6. **打包配置** - 添加 electron-builder 打包脚本
7. **测试运行** - 启动应用验证功能

## 验证方式
1. 运行 `npm start` 启动应用
2. 验证：点击开始 → 倒计时运行 → 计时结束弹出通知
3. 验证：专注/短休息/长休息自动切换
4. 验证：暂停、重置按钮功能正常
5. 验证：系统托盘最小化/恢复
6. 验证：番茄计数正确递增
